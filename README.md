## How to run

Solutions in multiple languages. To run them, you must go inside the specific
language directory.

### Go

Version used: `go1.19.3`.

```sh
$ go run day01/main.go
```

### Ruby

Version used: `ruby 3.1.2p20`.

```sh
$ ruby day01/main.rb
```
